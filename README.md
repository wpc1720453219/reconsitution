### 1.6
拆分阶段（154）:将逻辑分成两部分：一部分计算详单所需的数据，另一部分将数据渲染成 文本或HTML
第一阶段会创建一个中转数据结构，再把它传递给第二阶段。
总的来说： 过去在**印详单里调用相关的逻辑数据计算直接呈现出来**,
1.6 先进行计算出来将处理后的数据封装到实体类， 后通过调用文本或HTML方法呈现出来（将数据 与视图呈现分开）
